Replace placeholder images with optimized local assets.
Prefer AVIF/WebP with PNG/JPG fallbacks.
Always set explicit width/height or use aspect-ratio.
Keep file names stable to avoid broken links.
Optimize thumbnails for quick LCP and CLS stability.
Document sources and update image-sitemap.xml.
